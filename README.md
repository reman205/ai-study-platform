# 📚 AI Study Platform

AI-powered study platform for university students with personalized learning tools, AI tutoring, quiz generation, and smart summaries.

## 🌐 Live Demo

[Open Project](https://reman205.github.io/ai-study-platform/)

## ✨ Features

- Personalized student dashboard
- AI tutor for explanations and study tips
- Quiz generator
- Smart summarizer
- Flashcards and mind map support
- Clean educational UI

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript
